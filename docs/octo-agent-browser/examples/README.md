# Octo Agent Browser Examples

This directory contains example usage of the Octo Agent Browser library.

## Basic Example

```bash
bun run examples/basic_example.ts
```

Demonstrates:
- Creating an agent browser instance
- Navigating to a website
- Getting page title and text content
- Taking screenshots
- Clicking elements
- Filling forms
- Submitting forms

## Features

- Automated web browsing
- Form filling
- Screenshot capture
- Element interaction
- Page navigation
- Headless and non-headless modes
